# IAmLegend
《我是传奇》---一款小型的平行实境游戏。
#故事概述
在这个陌生或熟悉的城市里，到处都有漂泊的身影在穿梭，也许有团结组织，也许孤军奋战。行走在路上，我们不知道哪里是危险，哪里是安全，我们只能小心翼翼地向前。不能停止渴望，不能停止向前，这就是我们的传奇。
#设计原型

旨在
* 精简操作

* 提高互动反馈

* 体现显著的内在奖励

* 鼓励有明确目标的集体活动
